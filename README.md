# New Accents Photography Style Switcher

## Overview
This repository contains a photography studio website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website promotes a fictional wedding photography business and demonstrates dynamic website styling through multiple interchangeable themes. The project showcases advanced CSS techniques, responsive layouts, theme customization, and JavaScript-based style switching functionality.

## Features
* Wedding photography business website
* Dynamic style switching functionality
* Multiple visual themes and layouts
* Theme customization using CSS
* Interactive user experience
* Navigation menu system
* Photography portfolio presentation
* Contact and business information display

## Technologies Used
* HTML5
* CSS3
* JavaScript
* Dynamic Stylesheets
* CSS Visual Effects

## Project Files
* `na_home_txt.html` – Main website page
* `na_base.css` – Base stylesheet
* `na_layout.css` – Website layout and structure
* `na_style_0.css` – Joyous theme
* `na_style_1.css` – Elegant theme
* `na_style_2.css` – Innovative theme
* `na_style_3.css` – Romantic theme
* `na_style_4.css` – Zany theme
* `na_styler_txt.js` – Theme switching functionality
* Photography and branding image assets
